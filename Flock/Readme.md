Author: Xilu Zeng

How to Run

Open the project in Unity.

Press Play to start the simulation.

Click FlockManager to view the inspector window and play!

Other Controls
Key / Mouse	Action
W / A / S / D / Q / E	Move camera target
Mouse Wheel	Zoom in / out
R/Buttons on UI	Toggle auto camera rotation
Space	Scatter the flock temporarily
Notes

The top UI shows current status, trail mode, active flocking behaviors, and camera controls.

“Auto-Rotate: ON/OFF” indicates whether the camera is rotating automatically.
